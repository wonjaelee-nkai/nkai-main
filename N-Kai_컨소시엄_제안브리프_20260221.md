## AI 기반 보이스피싱 공동 대응 플랫폼 구축 (과기정통부/KISA 2026)

**제안 기업:** 뉴린카이로스에이아이(주) (N-Kai) | 대표 이원재  
**과제번호:** 2026-0172 | **마감:** 2026.03.16(월) 14:00

## 1. 제안 배경

본 과제는 경찰청·KISA·기업이 보이스피싱 관련 데이터를 실시간 수집·분석·공유하여 선제적으로 대응하는 플랫폼 구축을 목표로 합니다. N-Kai는 **행동 예측 AI 엔진 + 프라이버시 보호 기술** 파트를 담당하고, 귀사의 기존 탐지/분석 기술과 결합하여 **탐지(Detection) + 예측(Prediction)** 완전체 컨소시엄을 제안드립니다.

---

## 2. N-Kai가 가져오는 것

### 핵심 기술 — 특허 4건 우선심사 중

| 알고리즘 | 본 과제 적용 | 역할 |
|----------|-------------|------|
| **베이지안 추론** (확률 갱신) | 피해 확률 동적 갱신 | 통화·행동 데이터 기반 보이스피싱 피해 확률 실시간 산출 |
| **ROC 기반 임계치 보정** | 위험 임계점 동적 조정 | 연령·시간대·패턴별 최적 경보 기준선 자동 보정 |
| **비지도학습** (숨긴 패턴) | 신종 수법 선제 탐지 | 기존 시나리오에 없는 이상 행동 클러스터 자동 발견 |
| **Neural CDE** (시계열 시뮬레이션) | 피해 60분 전 사전 경보 | 불규칙 시계열 데이터의 연속 미분방정식 기반 궤적 예측 |

### 차별화 기술 — 온디바이스 ZKP(영지식증명)

- 통화 내용·행동 데이터를 **서버 전송 없이 단말 내에서 분석**
- 개인정보보호법 제약을 기술적으로 해결 (현행 보이스피싱 AI의 최대 병목)
- **과제 평가 시 "프라이버시 보호" 항목에서 결정적 차별점**

### 사업책임자 역량

- **이원재 대표** | 금융 FDS(이상거래탐지시스템) **20년 실전 경력**
- 은행·카드사 대상 사기탐지 시스템 설계·구축·운영
- 3년간 **70억원** 규모 프로젝트 성과 달성
- 5.5개월간 자기 데이터 기반 라이브 PoC 수행 중 (97.3% 내러티브 복원율 달성)

## 3. 제안 컨소시엄 구조

```
┌─────────────────────────────────────────────────┐
│          AI 보이스피싱 공동 대응 플랫폼           │
├────────────────────┬────────────────────────────┤
│   귀사 (주관/참여)  │     N-Kai (주관/참여)       │
│                    │                            │
│  ▸ 음성/텍스트 탐지  │  ▸ 행동 예측 AI 엔진        │
│  ▸ 딥보이스 판별    │  ▸ Neural CDE 시계열 예측   │
│  ▸ 기존 금융사/통신사 │  ▸ ZKP 온디바이스 프라이버시  │
│    네트워크 활용    │  ▸ 베이지안 위험도 산출       │
│  ▸ 데이터 수집·분석  │  ▸ 골든타임 60분 사전경보    │
└────────────────────┴────────────────────────────┘
     현재 위협 탐지          미래 위협 예측
     (Detection)           (Prediction)
```

**역할 분담은 유연하게 협의 가능합니다.**  
- N-Kai가 주관, 귀사가 참여 (또는 그 반대)  
- 참여 비율 10%~50% 범위 내 조정 가능  
- 핵심: 귀사의 **탐지 기술**과 N-Kai의 **예측 기술**이 결합되어야 과제 목표 달성

## 4. 귀사에 드리는 가치
