## 계약서 자동 생성 템플릿

### B2B SDK 라이선스 계약서 핵심 조항
1. 라이선스 범위 (API 호출 건수 기반)
2. 데이터 보안 및 개인정보 처리
3. 지식재산권 귀속 (N-KAI 특허 4건)
4. 성과 기반 정산 (ROI 달성 시 인센티브)
5. 계약 해지 및 위약금

### 투자 계약서 핵심 체크포인트
1. 지분율 및 희석 방지 조항
2. 우선청산권 (Liquidation Preference)
3. 동반매도권/태그얼롱
4. 이사회 구성 및 의결권
5. 정보 공개 의무

### NDA 비밀유지계약서 핵심 조항
1. 기밀정보 범위 (알고리즘, 특허, 고객데이터)
2. 유효기간 (계약 후 3년)
3. 위반 시 손해배상
4. 역엔지니어링 금지

## 법무 자동화 규칙
1. 계약서 검토 요청 시 → 핵심 독소조항 자동 감지
2. 소송 진행 상황 → Notion 자동 업데이트
3. 내용증명 수신 시 → 즉시 대응 전략 보고
4. 정부과제 계약 → 표준 계약서 자동 검토
5. 투자 계약 → 핵심 조항 체크리스트 자동 생성
