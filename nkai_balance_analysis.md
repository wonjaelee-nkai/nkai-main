## 3. calibrateVectors 보정 (아키타입 분류 개선)

### Before: 木과 火가 동일 보정 → NF 편중
```
木: N +0.2, F +0.2 → NF 비전가로 쏠림
火: N +0.2, F +0.2 → NF 비전가로 쏠림 (木과 구분 불가)
```

### After: 1차/2차 축 차등 보정
```
木: N +0.20 (1차), F +0.10 (2차) → N이 주도, 항해사(NT) 가능성 ↑
火: N +0.10 (2차), F +0.20 (1차) → F가 주도, 비전가(NF) 가능성 ↑
土: S +0.20 (1차), T +0.10 (2차) → S가 주도, 실용주의(SF) 가능성 ↑
金: S +0.10 (2차), T +0.20 (1차) → T가 주도, 분석가(ST) 가능성 ↑
水: N +0.15,       T +0.15       → 중립적, KIPA에 의해 결정
```

**효과: 같은 木/火도 다른 아키타입 그룹에 분산됨**
**의미: "같은 성장 에너지라도 木은 전략적, 火는 감성적"**

## 4. Phase 1 → Phase 2 전환 로드맵

### Phase 1 (2026.03~) N-Score 가중치
```
선천 기질 (사주)     40%  ← elementBase 345 균등
KIPA 성향            35%  ← 16문항 응답
공명 매트릭스         10%  ← 5×4 균등 매트릭스
골든타임 변동          10%  ← 월간 천간지지 변화
오행 엔트로피          5%  ← 5행 균형도
```

### Phase 2 (MyData 후) N-Score 가중치 변화
```
가입 첫 달:
  선천 40% + KIPA 35% + 행동 0% + 골든타임 20% + 엔트로피 5%

3개월 후:
  선천 30% + KIPA 25% + 행동 25% + 골든타임 15% + 엔트로피 5%

6개월 후:
  선천 25% + KIPA 20% + 행동 40% + 골든타임 10% + 엔트로피 5%

12개월 후:
  선천 15% + KIPA 10% + 행동 60% + 골든타임 10% + 엔트로피 5%
```

**핵심 원칙: 행동 데이터가 축적될수록 선천+설문의 비중 감소**
**유저 메시지: "N-KAI는 운명을 고정하지 않습니다. 행동이 DNA를 바꿉니다."**

### Phase 2 행동 데이터 점수 구성
```

리스크 관리 일관성 (riskConsistency)  × 150
포트폴리오-아키타입 정합도             × 100
행동 일관성 (동일 상황 동일 결정)      × 100
```

## 5. 밸런싱 후 예상 효과

### N-Score 분포 변화
```
Before (火 편향):
  火 일간: 평균 620, 중앙값 N5
  土 일간: 평균 520, 중앙값 N6
  → 1등급 차이 발생

After (균등):
  전 오행: 평균 570-590
  → 차이는 KIPA 응답과 공명 조합에 의해 결정
  → 같은 오행이라도 KIPA에 따라 ±100점 변동
```

### 16분류 분포 변화
```
Before:
  木/火 → NF 비전가 편중 (50%+ 예상)
  金/土 → ST 분석가 편중

After:
  木 → NT 항해사 가능성 ↑ (N 1차 보정)
  火 → NF 비전가 가능성 ↑ (F 1차 보정)  
  土 → SF 실용주의 가능성 ↑ (S 1차 보정)
  金 → ST 분석가 가능성 ↑ (T 1차 보정)
  水 → 균등 분산 (KIPA 의존)
  → 4개 그룹에 골고루 분산
```
