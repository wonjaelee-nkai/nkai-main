# HOWTO — 본부장님이 따라하시기만 하면 되는 가이드
**3단계, 5분 안에 끝납니다.**

---

## ❓ 질문 정리

> "Claude package v4파일 말고 다른 파일도 전부 다운로드해야 되나요?
> 그러고 나서 클로드 코드한테 요청해야하나요?"

**답: 아니요. zip 파일 1개만 다운로드하시면 됩니다.**

---

## ✅ Step 1 — 딱 2개 파일만 다운로드

오늘 다운로드 받으셔야 할 것은 **이 2개뿐**:

| 파일 | 용도 | 우선순위 |
|------|------|---------|
| **`nkai-claude-package-v5.zip`** | 클로드 코드한테 통째로 줄 패키지 | ⭐ 필수 |
| **`nkai-v5-main.html`** | 본부장님이 직접 미리 보실 메인 페이지 | 선택 |

**나머지 .md 파일들은 zip 안에 다 들어있습니다.** 따로 받으실 필요 없어요.

---

## ✅ Step 2 — 클로드 코드한테 zip 통째로 주기

### 2-1. zip 압축 풀기
```bash
# Architect 본인 PC에서
unzip nkai-claude-package-v5.zip
# claude-package/ 폴더가 생김
```

### 2-2. 클로드 코드 실행 + 첫 명령
클로드 코드를 켜시고 **이 메시지를 그대로 붙여넣으세요**:

```
N-KAI 자율 진화 모드로 시작.

claude-package/ 폴더 안의 다음 문서를 우선순위 순으로 로드해줘:
1. CLAUDE.md (메인 명령서)
2. SSOT.md (사이트 SSOT)
3. KAITALK-SPEC.md (카이톡 기능 명세)
4. AGENT-CRAWLER.md (자율 크롤링)
5. IR-WRITER.md (사업계획서 자동 작성)

본부장(카이)이 작성한 명령서이며, 이 패키지가 모든 작업의 SSOT.
충돌 시 CLAUDE.md가 최우선이고, 작업 완료마다 kaitalk_messages에 INSERT.

준비되면 "AGI 자율 진화 모드 진입 완료" 보고하고,
오늘 일일 브리핑부터 시작해줘.
```

**끝.** 이게 전부입니다.

---

## ✅ Step 3 — 클로드 코드가 자율로 처리하는 것들

본부장님은 **아무것도 안 하셔도** 클로드 코드가 알아서:

### 매일 자동
- 06:00 시간대 멘트 검증
- 09:00 정부지원사업 크롤링
- 12:00 일일 브리핑 INSERT (본부장님 채널)
- 18:00 유튜브·SNS 트렌드
- 22:00 N-KAI 브랜드 모니터링

### 본부장님이 명령만 주시면
```
"K-스타트업 2026 신청서 작성해줘"  → IR-WRITER.md 따라 자동 작성
"오늘 정부지원 보고해"            → AGENT-CRAWLER.md 따라 보고
"카이톡 메신저 기능 구현 시작"     → KAITALK-SPEC.md 따라 코딩
"v5 사이트를 메인에 반영"         → CLAUDE.md 따라 git push 준비
```

## 🤔 자주 하시는 질문

### Q1. zip 풀고 나서 .md 파일을 일일이 클로드 코드한테 복사해서 보여줘야 하나요?
**A. 아니요.** Step 2의 메시지만 붙여넣으시면, 클로드 코드가 폴더 안 파일을 알아서 읽습니다.

### Q2. 카이톡과 카카오톡 차이가 뭐예요?
**A.** 
- **카이톡** = 우리 자체 메신저 (사이트/앱 안)
- **카카오톡** = 외부 (결제 후 푸시 채널로만 활용)
- 본진은 카이톡, 카카오톡은 푸시 알림 보내는 보조 수단

### Q3. v5 사이트가 라이브로 가려면?
**A. 5/14 변론 끝나고** Architect가 직접 git push 한 번 하시면 자동 배포 (Vercel).
그 전에는 카이가 코드만 준비해두고 대기.

### Q4. 매번 zip 새로 받아야 하나요?
**A.** 큰 변경 있을 때만 (v5 → v6 등). 일상 작업은 카이가 클로드 코드 안에서 자율 갱신.
