## 3. ⭐ 4대 페르소나 그룹 SSOT (v6 신규)

### 3.1 그룹 명칭 (이름 변경 금지 — 본부장님 직접 지정)

| # | 그룹명 | 아이콘 | 태그라인 | 우세 자질 | 아키타입 |
|---|-------|--------|---------|----------|---------|
| 1 | **분석가 그룹** | ⚖️ | 신중하게 결정하는 사람 | 판단력·위기대응력 | #01~#04 |
| 2 | **실용주의 그룹** | 🔥 | 현실에 발 딛고 행동하는 사람 | 실행력·경제감각 | #05~#08 |
| 3 | **항해사 그룹** | 🧭 | 흐름을 읽고 키를 잡는 사람 | 4가지 균형형 | #09~#12 |
| 4 | **비전가 그룹** | 🌌 | 보이지 않는 것을 먼저 보는 사람 | 직관·위기 감지 | #13~#16 |

### 3.2 ❌ 금지 명칭 (사용 금지)
- 실행가 그룹 ❌ → **실용주의 그룹** ✅
- 균형가 그룹 ❌ → **항해사 그룹** ✅
- 직관가 그룹 ❌ → **비전가 그룹** ✅

### 3.3 그룹별 정의 SSOT

**분석가 그룹** ⚖️
- 정의: 데이터로 결정하고, 위험을 미리 보는 사람
- 특징: 안정적인 자산 형성과 장기 계획에 강함
- 강점: 큰 실수가 적음

**실용주의 그룹** 🔥
- 정의: 머리보다 현실. 실행 가능한 것만 골라 빠르게 움직이는 사람
- 특징: 창업·전직·투자에서 기회를 만들어냄
- 강점: 시기를 놓치지 않음

**항해사 그룹** 🧭
- 정의: 4가지 자질이 고르게 발달, 상황에 맞춰 방향을 조정하는 사람
- 특징: 어떤 시기든 키를 놓지 않음
- 강점: 흐름을 읽고 능동적으로 항해함
- ⭐ Architect 본부장님이 위치한 그룹 (예시 데모용)

**비전가 그룹** 🌌
- Definition: 미래를 먼저 보고, 직관과 통찰로 방향을 제시하는 사람
- 특징: 위험 신호를 가장 먼저 감지, 변동성 속에서 페이스 유지
- 강점: 미래를 먼저 그림

## 4. 4대 자질 SSOT (3개 아닌 4개 — 본부장님 직접 지정)

| 자질 | 아이콘 | 정의 |
|------|--------|------|
| **판단력** | ⚖️ | 데이터로 결정하는 능력 |
| **경제감각** | 💎 | 자산을 보는 직관 |
| **실행력** | 🔥 | 기회를 잡는 속도 |
| **위기대응력** | 🛡️ | 위험을 미리 보는 힘 |

### 4.1 ❌ 금지 표현
- *"핵심 자질 3개"* ❌ → *"4대 자질"* ✅
- *"분석적 사고 / 위기 직감 / 인내력"* (3개 태그) ❌ → 4대 자질 그대로 ✅

### 4.2 자질 → 그룹 매핑

```
분석가 그룹    = 판단력 高 + 위기대응력 高
실용주의 그룹  = 실행력 高 + 경제감각 高
항해사 그룹    = 4가지 모두 70~90 균형형
비전가 그룹    = 위기대응력 高 + 직관(특수) 高
```

## 9. 시그니처 ID 포맷 SSOT

```
NK-{NSCORE_4DIGIT}-{STAGE}{DOMAIN}-{YEAR}
예: NK-7261-S4D04-2026
```

- `NK` = N-KAI 식별자 (고정)
- `7261` = 사용자 N-Score 4자리
- `S4D04` = 단계·도메인 좌표
- `2026` = 발급 연도

활용처: 명함, PDF 리포트, SNS 공유, 카카오 채널
