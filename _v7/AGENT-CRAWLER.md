## 1. 정부지원사업 크롤링

### 1.1 대상 사이트
| 사이트 | URL | 빈도 |
|--------|-----|------|
| BizInfo (중기부) | bizinfo.go.kr | 일일 |
| K-Startup | k-startup.go.kr | 일일 |
| 창업진흥원 | kised.or.kr | 주 2회 |
| AI 허브 | aihub.or.kr | 주 1회 |
| 정보통신산업진흥원 (NIPA) | nipa.kr | 주 1회 |

### 1.2 검색 키워드 (필터)
**필수 매치 (둘 중 하나):**
- "AI" / "인공지능" / "딥러닝" / "머신러닝"
- "데이터" / "빅데이터" / "행동분석"

**가산점 매치 (있으면 우선순위 ↑):**
- "온디바이스" / "특허" / "초기창업"
- "B2C" / "구독" / "SaaS"
- "금융" / "헬스케어" / "라이프케어"

**제외:**
- "제조업" 단독
- "지역특화" (서울·경기 외)
- "5인 이상 매출" 같은 자격 미달 조건

### 1.3 일일 출력 포맷
```markdown
## 정부지원사업 일일 브리핑 [YYYY-MM-DD]

### 🔥 즉시 검토 (마감 D-7 이내)
1. [공고명] / [기관] / [지원금] / [마감일]
   - 적합도: ⭐⭐⭐⭐⭐
   - 핵심 요건: ...
   - URL: ...

### ⚠️ 주의 검토 (마감 D-30 이내)
...

### 💤 모니터링 (D-30 초과)
...
```

### 1.4 자동 액션
- ⭐⭐⭐⭐⭐ 적합도 → kaitalk_messages INSERT (severity=info, category=gov_funding)
- 마감 D-3 이내 → severity=warn으로 알림
- 마감 D-1 → severity=critical, 본부장 즉시 알림

## 2. 유튜브 트렌드 크롤링

### 2.1 대상 채널 (구독 모니터링)
- AI 관련: 슈카월드, 김미경TV, EO, 안될공학
- 핀테크: 토스피드, 카카오뱅크, NH투자증권
- 트렌드: 1분과학, 셜록현준

### 2.2 일일 검색 키워드
- "온디바이스 AI" / "Behavioral Prediction"
- "FDS" / "이상거래탐지"
- "라이프 OS" / "초개인화"
- "사주 AI" / "운세 AI" (B2C 시장 동향)

### 2.3 추출 데이터
- 제목, 채널명, 조회수, 업로드일
- 주요 댓글 5개 (감성 분석)
- 썸네일 패턴 (디자인 트렌드 참고)

### 2.4 자동 분석
- 조회수 10만+ 영상 → 상세 분석 보고서 작성
- 동일 키워드 3회 이상 출현 → 트렌드 보고
- N-KAI 경쟁 서비스 멘션 → 즉시 알림

## 5. 일일 통합 보고서 (kaitalk_messages)

### 5.1 INSERT 시각: 매일 12:00 KST

```sql
INSERT INTO kaitalk_messages (
  id, severity, category, architect_only,
  title, body, completed_at
) VALUES (
  gen_random_uuid(),
  'info',
  'daily_crawler_briefing',
  true,
  '카이 일일 브리핑 [YYYY-MM-DD]',
  '<<통합 보고서>>',
  NOW()
);
```
